# UI 风格规范 — 微信简洁风

## 设计哲学

克制、安静、信息优先。不用阴影堆层次，用留白和分割线建立节奏。让内容本身成为视觉焦点，UI 退到幕后。

## 配色

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景色 | `#EDEDED` | 页面底色，微灰 |
| 卡片/列表背景 | `#FFFFFF` | 纯白 |
| 主文字 | `#181818` | 接近纯黑 |
| 副文字 | `#666666` | 灰色 |
| 辅助文字 | `#999999` | 浅灰 |
| 分割线 | `#E5E5E5` | 极淡灰 |
| 品牌色/强调色 | `#07C160` | 微信绿，仅用于按钮和关键操作 |
| 链接色 | `#576B95` | 微信蓝，用于可点击文字 |
| 危险色 | `#FA5151` | 红色，仅用于删除/错误 |

### 使用原则

- 大面积用白底 + 灰底，品牌绿只在按钮和极少数强调处出现
- 不用渐变、不用彩色背景
- 深色模式暂不考虑

## 字体

| 层级 | 大小 | 粗细 | 颜色 | 用途 |
|------|------|------|------|------|
| 标题 | 17sp | w600 | #181818 | 页面标题、列表项主文字 |
| 正文 | 15sp | w400 | #181818 | 内容正文 |
| 副文 | 13sp | w400 | #666666 | 描述、时间、作者 |
| 辅助 | 11sp | w400 | #999999 | 标签、提示 |

### 原则

- 不用花哨字体，系统默认即可
- 行高 1.4 ~ 1.5
- 标题不超过一行，超出省略号

## 间距

| 名称 | 值 | 用途 |
|------|-----|------|
| 页面水平边距 | 16px | 内容区左右留白 |
| 列表项内间距 | 16px | 列表项内部 padding |
| 列表项间距 | 0 | 列表项之间无间距，用分割线分隔 |
| 段落间距 | 12px | 上下内容块之间 |
| 图标与文字间距 | 12px | 头像/图标和右侧文字 |

## 列表项

微信风格列表项结构：

```
┌────────────────────────────────────────┐
│ [图标 40x40]  标题文字          [箭头>] │
│               副文字/描述              │
├────────────────────────────────────────┤  ← 分割线（左侧缩进 68px）
│ [图标 40x40]  标题文字          [箭头>] │
│               副文字/描述              │
└────────────────────────────────────────┘
```

### 关键规则

- **无卡片阴影**：列表项之间用 0.5px 分割线，不用 Card 的 elevation
- **分割线左缩进**：从图标右侧开始，不顶到最左边
- **白色背景**：列表区域纯白底
- **点击态**：轻微灰色高亮（#F5F5F5），不用 ripple 水波纹
- **右侧箭头**：chevron_right 图标，颜色 #C7C7CC
- **图标**：40x40 圆角矩形（borderRadius: 6），无边框

## 页面结构

### 导航栏（AppBar）

- 背景色：`#EDEDED`（与页面底色一致）或 `#FFFFFF`
- 无阴影（elevation: 0）
- 标题居中，17sp w600 黑色
- 返回按钮：左箭头，无文字

### 页面底色

- 用 `#EDEDED` 作为 Scaffold 背景
- 内容区（列表/卡片）用白色块
- 白色块之间用 8px 灰色间隔（模拟微信的分组效果）

### 分组

多个列表项组成一个"组"，组与组之间留 8px 灰底间隔：

```
[灰底 8px]
┌─ 白色组 ─────────────────────┐
│ 列表项 1                      │
│ ─────────── 分割线 ────────── │
│ 列表项 2                      │
│ ─────────── 分割线 ────────── │
│ 列表项 3                      │
└───────────────────────────────┘
[灰底 8px]
┌─ 白色组 ─────────────────────┐
│ 列表项 4                      │
└───────────────────────────────┘
```

## 按钮

| 类型 | 样式 |
|------|------|
| 主按钮 | 绿色填充（#07C160），白色文字，圆角 6px，高度 44px |
| 次要按钮 | 白色填充，绿色文字，绿色边框 1px |
| 文字按钮 | 无背景，链接蓝（#576B95）文字 |

- 按钮宽度跟随内容或撑满（取决于场景）
- 禁用态：opacity 0.5

## 标签/Badge

- 小圆角矩形（borderRadius: 3）
- 背景 `#F5F5F5`，文字 `#666666`
- 内间距 horizontal: 6, vertical: 2
- 不用彩色标签

## 图片

- 列表中的图标：40x40，圆角 6px
- 详情页头图：无特殊处理，保持原始比例
- 加载占位：纯灰色块（#F5F5F5）
- 加载失败：灰色块 + 默认图标

## 动画

- 克制使用，不加花哨动画
- 页面切换：默认 MaterialPageRoute 右滑入
- 点击反馈：背景变灰（150ms），不用缩放/弹跳
- 列表加载：底部小 loading 指示器，不用骨架屏

## 禁止事项

- ❌ 不用 Card 的 elevation/阴影
- ❌ 不用渐变色
- ❌ 不用 Material 水波纹（ripple），用背景色变化代替
- ❌ 不用大面积彩色
- ❌ 不用圆形头像（用圆角矩形）
- ❌ 不用浮动按钮（FAB）
- ❌ 标题不用全大写
